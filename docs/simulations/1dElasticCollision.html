A simple elastic collision.

Two masses move in opposite direction and collide which each other. Each mass has a initial velocity that changes after the collision.

The [Reset] button will reset the simulation and apply any values you may have changed.

Note that the collision time is not always accurate.

The Math

Final velocities

Because this is an elastic collision, there are 2 conservation laws they blocks must follow.

Conservation of momentum:

m_1 v_1 + m_2 v_2 = m_1 v'_1 + m_2 v'_2

Conservation of kinetic energy:

\frac{1}{2}m_1 v_1^2 + \frac{1}{2}m_2 v_2^2 = \frac{1}{2}m_1 v_1^{\prime 2} + \frac{1}{2}m_2 v_2^{\prime 2}

Our unknowns are v'_2 and v_1'. There are 2 equations and 2 unknowns, so we can begin solving immediately

Cancel \frac 12s

m_1 v_1^2 + m_2 v_2^2 = m_1 v_1^{\prime 2} + m_2 v_2^{\prime 2} m_1 v_1^2 - m_1 v_1^{\prime 2} = m_2 v_2^{\prime 2} - m_2 v_2^2 m_1( v_1^2 - v_1^{\prime 2} ) = m_2 ( v_2^{\prime 2} - v_2^2 )

Rearrange conservation of momentum

m_1 v_1 - m_1 v'_1 = m_2 v'_2 - m_2 v_2 m_1( v_1 - v'_1 ) = m_2 ( v'_2 - v_2 )

Divide the two equations

\frac{m_1( v_1^2 - v_1^{\prime 2} )}{m_1( v_1 - v'_1 )} = \frac{m_2 ( v_2^{\prime 2} - v_2^2 )}{m_2 ( v'_2 - v_2 )} \frac{\cancel{m_1} ( v_1^2 - v_1^{\prime 2} )}{\cancel{m_1} ( v_1 - v'_1 )} = \frac{\cancel{m_2} ( v_2^{\prime 2} - v_2^2 )}{\cancel{m_2} ( v'_2 - v_2 )} \frac{ v_1^2 - v_1^{\prime 2} }{ v_1 - v'_1 } = \frac{ v_2^{\prime 2} - v_2^2 }{ v'_2 - v_2 }

Expand difference of squares

\frac{ (v_1 - v_1')(v_1 + v_1') }{ v_1 - v'_1 } = \frac{ (v_2' - v_2)(v_2' + v_2) }{ v'_2 - v_2 } \frac{ \cancel{(v_1 - v_1')}(v_1 + v_1') }{ \cancel{v_1 - v'_1} } = \frac{ \cancel{(v_2' - v_2)}(v_2' + v_2) }{ \cancel{v'_2 - v_2} } v_1 + v_1' = v_2' + v_2

Solve for v_1'

v_1' = v_2' + v_2 - v_1

Substitute into conservation of momentum

m_1 v_1 + m_2 v_2 = m_1 (v_2' + v_2 - v_1) + m_2 v'_2 m_1 v_1 + m_2 v_2 = m_1 v_2' + m_1 v_2 - m_1 v_1 + m_2 v'_2 m_1 v_1 + m_2 v_2 = m_1 v_2 - m_1 v_1 + (m_1 + m_2) v'_2 m_1 v_1 + m_2 v_2 - m_1 v_2 + m_1 v_1 = (m_1 + m_2) v'_2 \frac{ m_1 v_1 + m_2 v_2 - m_1 v_2 + m_1 v_1 }{ m_1 + m_2 } = v'_2 v'_2 = \frac{ 2 m_1 v_1 + m_2 v_2 - m_1 v_2 }{ m_1 + m_2 }

Rearrange

v'_2 = \frac{ 2 m_1 v_1 }{ m_1 + m_2 } + \frac{m_2 v_2 - m_1 v_2 }{ m_1 + m_2 } v'_2 = \frac{ 2 m_1 v_1 }{ m_1 + m_2 } + \frac{m_2 - m_1 }{ m_1 + m_2 } v_2

Returning and solve for v_2'

v_1 + v_1' = v_2' + v_2 v_1 + v_1' - v_2 = v_2'

Substitute into conservation of momentum

m_1 v_1 + m_2 v_2 = m_1 v'_1 + m_2 v'_2 m_1 v_1 + m_2 v_2 = m_1 v'_1 + m_2 (v_1 + v_1' - v_2) m_1 v_1 + m_2 v_2 = m_1 v'_1 + m_2 v_1 + m_2 v_1' - m_2 v_2 m_1 v_1 + m_2 v_2 = m_2 v_1 + ( m_1 + m_2 ) v_1' - m_2 v_2 m_1 v_1 + m_2 v_2 + m_2 v_2 - m_2 v_1 = ( m_1 + m_2 ) v_1' \frac{m_1 v_1 + m_2 v_2 + m_2 v_2 - m_2 v_1}{ m_1 + m_2 } = v_1' \frac{m_1 v_1 + 2m_2 v_2 - m_2 v_1}{ m_1 + m_2 } = v_1'

Rearrange

\frac{ 2m_2 v_2 }{ m_1 + m_2 } + \frac{ m_1 v_1 - m_2 v_1 }{ m_1 + m_2 } = v_1' \frac{ 2m_2 v_2 }{ m_1 + m_2 } + \frac{ m_1 - m_2 }{ m_1 + m_2 } v_1 = v_1' v_1' = \frac{ 2m_2 v_2 }{ m_1 + m_2 } + \frac{ m_1 - m_2 }{ m_1 + m_2 } v_1

Point of collision

The equation to determine where the blocks would hit uses the kinematic equation

x' = vt + x

We have two blocks, so

x_0' = v_0 t + x_0 x_1' = v_1 t + x_1

We know that x_0' = x_1' when the collision occurs.

v_1 t + x_1 = v_0 t + x_0

Solve for t

v_1 t - v_0 t = x_0 - x_1 (v_1 - v_0) t = x_0 - x_1 t = \frac{x_0 - x_1}{v_1 - v_0}

And we can determine the position by substituting t into one of the equations we started with.

x_0' = v_0 t + x_0 x_0' = v_0 \cdot \frac{x_0 - x_1}{v_1 - v_0} + x_0
