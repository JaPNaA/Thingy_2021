Like 1d motion, but in 2d.

The ball moves in a trajectory as defined by its initial velocity and initial position.

Gravity is set to -9.8 \frac{m}{s^2} \textup{[down]} initially.

Holding [space] allows you to see its movement in real time. You can hover over the red dot to see it's position at any time. Note that in this simulation, positive y is in the downward direction.

The Math

The simulation uses one of the five kinematic equations. With the given information, initial velocity \vec v, time t, and acceleration \vec a, we can use it to find it's position at any point in time.

\vec d = \vec v t + \frac{1}{2}\vec at^2
